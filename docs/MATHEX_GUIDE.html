MESC-Mathex: Syntax & User Guide

Platform: MESC-Mathex IDE

Description: A comprehensive reference guide for array manipulation, plotting, and standard Mathex syntax.

Execution Shortcuts:
F5 : Run the entire active script.
F9 : Run the currently selected lines (or current line if no selection).

1. Core Environment

Mathex is built to process procedural array-based mathematics natively. The workspace consists of:

  • Editor: Write your multi-line .m scripts with syntax highlighting and auto-indentation.
  • Console: Directly evaluate fast mathematical commands and view execution outputs or error tracebacks.
  • Variable Inspector: Real-time tracking of all active workspace variables, matrices, and their dimensions.
  • Plot Dock: An embedded engine for rendering high-performance 2D and 3D graphs natively.

2. Basic Syntax & Matrix Operations

In Mathex, everything defaults to double-precision arrays.

Variable Assignment & Arrays

% Single values
a = 10;
b = pi * 2;

% Row vectors and Column vectors
row = [1, 2, 3, 4];
col = [1; 2; 3; 4];

% Matrix generation
A = [1, 2; 3, 4];
B = zeros(2, 2);
C = ones(3, 3);
I = eye(4); % Identity matrix

Element-wise vs Matrix Operations

Operation Matrix-wise Element-wise
MultiplicationA * BA .* B
DivisionA / BA ./ B
ExponentiationA ^ 2A .^ 2

3. Control Flow & Scripting

Logic blocks must be closed with the end keyword.

% If-Else Conditionals
x = 5;
if x > 10
    disp('Greater than 10');
elseif x == 5
    disp('Exactly 5');
else
    disp('Less than 10');
end

% For Loops (using colon operator for ranges)
for i = 1:1:10
    disp(i);
end

% While Loops
count = 0;
while count < 5
    count = count + 1;
end

4. Plotting & Graphics

Mathex handles rendering via the embedded Plot Dock. Use standard figure manipulation commands.

% 2D Line Plot
x = linspace(0, 2*pi, 100);
y = sin(x);

plot(x, y);
title('Sine Wave');
xlabel('Time (s)');
ylabel('Amplitude');
grid on;

% 3D Surface Plot
[X, Y] = meshgrid(-5:0.5:5, -5:0.5:5);
Z = X.^2 + Y.^2;
surf(X, Y, Z);

% 3D Animated Comet
t = 0:pi/50:10*pi;
comet3(sin(t), cos(t), t);
