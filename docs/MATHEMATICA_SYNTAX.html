MESC-Mathematica: Syntax & User Guide

Platform: MESC-Mathematica IDE

Description: A comprehensive reference guide for mathematical syntax, functions, and engine capabilities.

Execution Shortcut: To evaluate any cell in the notebook, press Shift + Enter.

1. Core Principles

MESC-Mathematica strictly mirrors traditional Wolfram syntax conventions. Please keep the following golden rules in mind:

  • Capitalization: All built-in functions and constants begin with a capital letter (e.g., Sin, Plot, Pi).
  • Square Brackets: Function arguments are always enclosed in square brackets [ ], not parentheses. Example: Cos[x].
  • Curly Braces: Lists, vectors, matrices, and ranges are enclosed in curly braces { }. Example: {1, 2, 3}.
  • Parentheses: Used strictly for grouping mathematical operations. Example: (x + 2) * y.

3. Algebra & Equation Solving

Use double equals == to define mathematical equalities, and a single equals = to assign variables.

(* Assigning an expression to a variable *)
eq = x^2 - 4 == 0

(* Solving the equation for x *)
Solve[eq, x]

(* Expanding algebraic expressions *)
Expand[(x + y)^3]

(* Simplifying complex expressions *)
Simplify[Sin[x]^2 + Cos[x]^2]

4. Calculus

MESC-Mathematica provides powerful symbolic calculus tools powered by the SymPy backend.

Differentiation

Use the D function. The syntax is D[expression, variable].

D[x^3 + 2x^2, x]          (* First derivative *)
D[Sin[x], {x, 2}]         (* Second derivative *)

Integration

Use the Integrate function. For definite integrals, provide a range list.

(* Indefinite Integral *)
Integrate[x^2, x]

(* Definite Integral from 0 to Pi *)
Integrate[Sin[x], {x, 0, Pi}]

Limits & Series

(* Limits: Use an arrow -> for the approach *)
Limit[Sin[x]/x, x -> 0]

(* Taylor/Maclaurin Series expansion up to order 5 *)
Series[Exp[x], {x, 0, 5}]

5. Matrices & Linear Algebra

Matrices are defined as lists of lists.

Function Syntax Description
Creation m = {{a, b}, {c, d}} Defines a 2x2 matrix.
Determinant Det[m] Calculates the determinant.
Inverse Inverse[m] Calculates the matrix inverse.
Eigenvalues Eigenvalues[m] Returns the eigenvalues of the matrix.

6. Plotting & Graphics

Visualization is fully native. You can easily generate 2D and 3D plots.

(* Standard 2D Plot *)
Plot[Sin[x], {x, -10, 10}]

(* Plotting multiple functions together *)
Plot[{Sin[x], Cos[x]}, {x, -Pi, Pi}]

(* 3D Surface Plot *)
Plot3D[Sin[x] * Cos[y], {x, -5, 5}, {y, -5, 5}]

7. Interactive Manipulation

The Manipulate function allows you to create highly interactive outputs with dynamic sliders, all processed natively without page-reloads.

(* Interactive Sine Wave changing frequency 'a' *)
Manipulate[
    Plot[Sin[a * x], {x, 0, 10}], 
    {a, 1, 5}
]

(* Interactive 3D Plot changing phase 'p' *)
Manipulate[
    Plot3D[Sin[x + p] * Cos[y], {x, -5, 5}, {y, -5, 5}], 
    {p, 0, 2*Pi}
]
